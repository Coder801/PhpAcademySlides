Базовый синтаксис CSS

Пример

			h1 {color: #fff;} // Задать элементу h1 белый цвет текста
			.info {background: red;} // Задать блоку info фон красного цвета

Способы добавления CSS правил

  1. Внутренние стили элемента
  2. Таблицы глобальных стилей
  3. Таблицы связанных стилей
  4. Импорт в таблице связанных стилей

Обращение к элементам

  1. по тегам (H1, TD, HR...);
  2. универсальный селектор — *
  3. селекторы по классу (.main, .header);
  4. селекторы по ID (#form, #submit);
  5. селекторы атрибутов ([data-name], [data-name$="hidden"]);
  6. псевдо-классы:
    • динамические (:hover, :focus);
    • селектор цели (:target);
    • селектор по языку (:lang(en));
    • состояния элементов интерфейса (:enabled, :checked);
    • структурные псевдоклассы (:root, :nth-child);
    • псевдокласс отрицания (:not(P)).
  7. псевдоэлементы (:before, :first-line);
  8. комбинаторы (DIV SPAN, P + H1);
