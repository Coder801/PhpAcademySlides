DOM (Document Object Model)

Объектная модель, используемая для XML/HTML-документов.

Документ является иерархией. Каждый HTML-тег образует отдельный элемент-узел, каждый фрагмент текста - текстовый элемент, и т.п.

Пример DOM

Типы узлов

1 ELEMENT_NODE Узел элемента
3 TEXT_NODE Текстовый узел (#text)
7 PROCESSING_INSTRUCTION_NODE Узел инструкции обработки
8 COMMENT_NODE Узел комментария (#comment)
9 DOCUMENT_NODE Узел документа (#document)
10 DOCUMENT_TYPE_NODE Узел типа документа
11 DOCUMENT_FRAGMENT_NODE Узел фрагмента документа

Типы узлов

nodeType — Определить тип узла

nodeValue — Строка, представляющая значение узла.

nodeName — Возвращает имя узла. Если узел является элементом, то свойство nodeName возвращает имя тега.

tagName — Возвращает имя тега (только у элементов)

Навигация по DOM

Самые верхние элементы дерева доступны напрямую из document.

HTML = document.documentElement

BODY = document.body

HEAD = document.head

Разделение DOM узлов

Дочерние – элементы, которые лежат непосредственно внутри данного..

Потомки – все элементы, которые лежат внутри данного, вместе с их детьми, детьми их детей и так далее. То есть, всё поддерево DOM.

Родитель – блок в котором находится выше уровня текущего элемента

Соседние – элементы находящиеся на одном уровне

Дочерние

element.childNodes - Список дочерних узлов

element.children - Список дочерних элементов

element.firstChild - Возьмет первый дочерний узел

element.firstChildElement - Возьмет первый дочерний элемент

element.lastChild - По аналогии возьмет последний узел

element.lastChildElement - По аналогии возьмет последний элемент

DOM-коллекции, такие как childNodes и другие. В них нет методов массивов, таких как forEach, map, push, pop и других.

Потомки, соседи

element.previousSibling - Предыдущий элемент в DOM

element.nextSibling - Следующий элемент в DOM

element.parentNode - Ссылка на узел родителя

element.parentElement - Ссылка на родителький элемент

Особые ссылки для таблиц

  • table.rows – коллекция строк TR таблицы.
  • table.caption/tHead/tFoot – ссылки на элементы таблицы CAPTION, THEAD, TFOOT.
  • table.tBodies – коллекция элементов таблицы TBODY.
  • tbody.rows – коллекция строк TR секции.
  • tr.cells – коллекция ячеек TD/TH
  • tr.sectionRowIndex – номер строки в текущей секции THEAD/TBODY
  • tr.rowIndex – номер строки в таблице
  • td.cellIndex – номер ячейки в строке

Выбор элементов

document.getElementById('intro') - Выбрать элемент по Id

getElementsByTagName('p') - Выбрать элемент по тегу

getElementsByClassName('intro') - Выбрать элемент по классу

querySelector(".list") - Выбор по CSS классу

querySelectorAll('.list > li') - Выбор массива элементов по CSS классу

closest('ul') - ищет ближайший элемент от текущего

matches('.someClass') - вернет true/false в случае если селектор найден

Свойства узлов

innerHTML - HTML находящийся внутри элемента

outerHTML - HTML с текущим элементом (только чтение)

nodeValue/data - Содержимое других узлов, текстовых или комментариев

textContent - Выводит текстовый узел

hidden - значение true скроет элемент

Атрибуты DOM элементов

elem.hasAttribute(name) – Проверяет наличие атрибута

elem.getAttribute(name) – Получает значение атрибута

elem.setAttribute(name, value) – Устанавливает атрибут

elem.removeAttribute(name) – Удаляет атрибут

elem.attributes - Получить массив аттрибутов

elem.dataset.delay - Получить значение data-атрибута

Атрибуты для работа с классами

elem.classList – Доступ к обьекту классов

elem.classList.add('className') – Добавить класс

elem.classList.remove('className') – Удалить класс

elem.classList.toggle('className') – Триггер класса

elem.classList.contains('className') - Вернет true если класс есть
