Структура HTML документа

			<!DOCTYPE html> <!-- Декларация -->
			<html lang="en"> <!-- Начало HTML документа -->
			<head> <!--Служебная информация-->
			    <title>HTML First page</title> <!-- Заголовок -->
			</head>
			<body>
				<h1>Hello world!<h1> 
			</body>
			</html>

DOCTYPE - указывает, в соответствии с каким стандартом HTML написана ваша Web-страница.

<!DOCTYPE html> Для всех документов.
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
"http://www.w3.org/TR/html4/strict.dtd">		 Строгий синтаксис HTML
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
"http://www.w3.org/TR/html4/loose.dtd">		 Переходный синтаксис HTML.
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN"
"http://www.w3.org/TR/html4/frameset.dtd">		 В HTML-документе применяются фреймы.

Форматирование текста

  • <strong> Выделеный текст</strong>

Гиперссылки

			<body>
				<a href="www.google.com" target="_blank">Google.com</a> 
      <!-- Ссылка на внешний ресурс (открывается в новом окне) -->
				<a href="/about.html">About</a> <!-- Ссылка на другую HTML-страницу -->
				<a href="#header">Back to top</a> <!-- Ссылка на ресурс внутри документа -->
				<a href="mailto:joe@example.com">Send mail</a> <!-- Почтовая ссылка -->
			</body>

Iframe

Cоздает плавающий фрейм, который находится внутри обычного документа, он позволяет загружать в область заданных размеров любые другие независимые документы.

					<!DOCTYPE html>
					<html lang="en">
					<head>
					 <iframe src="http://localhost:8080#slide20">
					  <p>browser does not support iframes.</p>
					 </iframe>
					</head>
