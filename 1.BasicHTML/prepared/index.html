Немного истории...

Язык HTML был разработан британским учёным Тимом Бернерсом-Ли приблизительно в 1986—1991 годах в стенах CERN в Женеве в Швейцарии

  • HTML 0.9;
  • HTML 2.0, одобренный как стандарт 22 сентября 1995 года;
  • HTML 3.0 — март 1995 года;
  • HTML 4.0 — 18 декабря 1997 года;
  • HTML 4.01 24 декабря 1999 года;
  • HTML5 — 20 ноября 2007 года;
  • HTML 5.1 начал разрабатываться 17 декабря 2012 года.

Пример простой страницы

			<!DOCTYPE html>
			<html lang="en">
			<head>
			    <title>HTML First page</title>
			</head>
			<body>
				<h1>Hello world!<h1> 
			</body>
			</html>

Структура HTML документа

			<!DOCTYPE html> <!-- Декларация -->
			<html lang="en"> <!-- Начало HTML документа -->
			<head> <!--Служебная информация-->
			    <title>HTML First page</title> <!-- Заголовок -->
			</head>
			<body>
				<h1>Hello world!<h1> 
			</body>
			</html>

DOCTYPE - указывает, в соответствии с каким стандартом HTML написана ваша Web-страница.

!DOCTYPE html Для всех документов.
!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
"http://www.w3.org/TR/html4/strict.dtd">		 Строгий синтаксис HTML
!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
"http://www.w3.org/TR/html4/loose.dtd"		 Переходный синтаксис HTML.
!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN"
"http://www.w3.org/TR/html4/frameset.dtd"		 В HTML-документе применяются фреймы.

Параграф, Форматирование текста

Тег <p> - определяет параграф

Форматирование текста

  • <strong> Выделеный текст </strong>

  • <em> Курсив </em>

  • <ins> Подчеркнет </ins> или <strike>перечернет</strike> текст

  • <sub> Нижний </sub> или <sup> верхний </sup> индекс

Гиперссылки

			<body>
				<a href="www.google.com" target="_blank">Google.com</a> 
      <!-- Ссылка на внешний ресурс (открывается в новом окне) -->
				<a href="/about.html">About</a> <!-- Ссылка на другую HTML-страницу -->
				<a href="#header">Back to top</a> <!-- Ссылка на ресурс внутри документа -->
				<a href="mailto:joe@example.com">Send mail</a> <!-- Почтовая ссылка -->
			</body>

Работа с изображениями

			<!DOCTYPE html>
			<html lang="en">
			<head>

			</head>
			<body>
				<img src="img/some-image.jpg" alt=""/> <!-- Изображение -->
			</body>
			</html>
